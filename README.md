## 🚀 Quick Start

**Installation**

```shell
  npm install
```

**Running server / development**

```shell
  npm run dev
```

**Running server / production**

```shell
  npm run build
  npm run start
```

<br/>

## 📚 Technologies

- NodeJS
- Express
- MongoDB
- TypeScript
- REST API
- JWT
- Eslint & prettier
